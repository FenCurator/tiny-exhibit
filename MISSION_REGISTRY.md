# Mission Registry

Mission Mode 的中央登錄。Mission 可以開新 repo，但狀態要回到這裡，避免做到一半失散。

## 狀態定義

- `active`：正在進行
- `completed`：任務本體完成
- `blocked`：所有免費可行路徑都有實測阻塞證據
- `incomplete`：暫存狀態，不可當成功回報
